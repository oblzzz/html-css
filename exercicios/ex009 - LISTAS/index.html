TRABALHANDO COM LISTAS

- Lista Ordenada

Conforme os itens vão sendo adicionados, a numeração é atualizada automaticamente.
Existem 5 formatos de listas ordenadas:
1. Números (1, 2, 3) - ol type="1"
2. Letras minúsculas (a, b, c) - ol type="a"
3. Letras maiúsculas (A, B, C) - ol type="A"
4. Números romanos (I, II, III) - ol type="I"
5. Números romanos minúsculos (i, ii, iii) - ol type="i"

- Listas não ordenadas

As listas não ordenadas são usadas quando a ordem dos itens não é importante.
Os itens são marcados com marcadores (bullets) em vez de números.
Existem 3 tipos de marcadores para listas não ordenadas:
1. Círculos preenchidos - ul type="disc"
2. Círculos vazados - ul type="circle"
3. Quadrados preenchidos - ul type="square"
