TAG REL

Quando usar o comando target= "_blank" para abrir um link em uma nova aba, é fortemente recomendado o uso do atributo rel="external noopener noreferrer"

target="_blank": Abre o link em uma nova aba

rel="external": Semântico. Indica que o link é externo

"noopener noreferrer": Evita vulnerabilidades de segurança ao abrir links em novas abas
